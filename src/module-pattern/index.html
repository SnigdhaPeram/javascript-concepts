Module Pattern

Definition

What is module pattern ?

Module pattern or Revealing Module pattern is good way to create an utility object where we can have private / public variables and methods.

When We need it ?

When we do not want to create the instance of the Class and just want to have only one instance then Module pattern is preferrable.

Example ( Please check console log for output of below example )

 
                function logger() {

                    /** private variable */
                    var version = 'Logger Version 2.3: ';
                
                    /**
                     * public functions
                     */
                    return {
                        info: info,
                        debug: debug
                    }
                
                    /** private function */
                    function info(msg) {
                        console.info(version + msg);
                    }
                
                    /** private function */
                    function debug(msg) {
                        console.debug(version + msg);
                    }
                }
                
                var log = logger();
                log.info('This is info message');
                log.info('This is debug message');
