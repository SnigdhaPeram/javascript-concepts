Hoisting

Definition
Hoisting is javascript priniciple where any varibales gets declared at the begining of the function.

Example ( Please check console log for output of below example )

 
        
var x = 5;
function print() {
  if(x==5) {
    var x = 6;
    console.log(x);
  } else {
    x = 7;
    console.log(x); 
  }
}
print() ; // what will be output ?
